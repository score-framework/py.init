.. _init_glossary:

.. glossary::

    confdict
        The input value to an ``init`` function. A `dict` mapping
        configuration keys to values. The :ref:`introduction` can give you a
        good example on its meaning and usage.

    configuration helper
        A class that can be used to define part of a module configuration in
        python code. The narrative documentation on the SCORE initialisation
        process contains a :ref:`section describing the use of configuration
        helpers <configuration_helper>` in detail.
